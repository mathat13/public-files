Steps to create new project:

- Add project name in CMakeLists.txt on below line (in CMakeLists.txt):

project(project_name VERSION 0.1.0 LANGUAGES C)

- Copy contents of this folder into new directory for new project

In preferred order:

- Add tests to tests.c
- Add functions to function.C
- Add headers to function.h

- Configure, build, and run!
